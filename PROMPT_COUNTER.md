# Prompt Counter

How many human prompts it took to build diffy — a native macOS git diff viewer
with side-by-side panes, intra-line highlights, a folder tree, a branch wizard,
fold-away unchanged lines, a .pkg installer, and a GitHub repo.

Count: 11

(Bumped automatically by a Claude Code hook on every prompt.)

## The prompts

1. Build an IntelliJ-style diff viewer for macOS, invokable as `diffy master develop`. "Come back when you have the finished diff viewer!"
2. How can I send this program to someone?
3. So it's not installed anymore on my system now?
4. Add a collapsible folder tree + a branch-selection wizard with local/origin toggle and search.
5. Can we make a .pkg installer with full uninstall instructions?
6. Would be cool if unchanged lines were collapsed (but expandable).
7. Check for security vulnerabilities, then push everything to GitHub.
8. Create this prompt counter.
