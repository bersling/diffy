What gets installed

  • /usr/local/bin/diffy — the diff viewer (universal: Apple Silicon & Intel)
  • /usr/local/share/diffy/uninstall.sh — uninstaller

Usage

Open Terminal inside any git repository:

diffy                    # branch-selection wizard
diffy master develop     # compare two branches
diffy origin/master origin/develop
diffy master...develop   # merge-base semantics (like a GitLab MR)
diffy --help             # everything else

How to FULLY uninstall

Run the bundled uninstaller:

sudo /usr/local/share/diffy/uninstall.sh

It removes everything diffy ever put on your system:

  • /usr/local/bin/diffy (the program)
  • /usr/local/share/diffy/ (this uninstaller)
  • the macOS installer receipt (pkgutil --forget com.taskbase.diffy)
  • ~/Library/Preferences/diffy.plist (remembered window size/position)

Or do the same by hand with the four commands above. diffy writes nothing else anywhere — no caches, no launch agents, no background services.
